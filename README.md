SWORD GROWER - 極限の剣士SWORD GROWER は、迫りくる敵をなぎ倒し、自身の剣を極限まで強化していくHTML5サバイバルアクションゲームです。ブラウザさえあれば、インストール不要ですぐにプレイ可能です。⚔️ ゲーム概要プレイヤーは、マウスで狙いを定め、強力な「斬撃」で敵を討伐します。敵が落とすオーブ（経験値）を集めてレベルアップし、プレイスタイルに合わせたスキルを選択して、3分間の生存を目指します。🌟 注目のシステム動的なスキルビルド: レベルアップ時に提示されるランダムな3つのスキルから1つを選択。「剛腕（攻撃力）」か「長剣（リーチ）」か、戦略はあなた次第です。魔法反射: 魔法使い（Mage）が放つ弾をタイミングよく斬ることで、敵に向かって跳ね返すことが可能です。群衆AI: 敵同士が重なり合わないように分離するロジックを搭載し、リアルな包囲網を再現しています。🕹 操作方法操作アクションマウス移動照準（攻撃方向）の変更左クリック斬撃（攻撃） / ゲームスタートWASD / 移動マウス方向への自動微進（マウスと自機の距離で制御）🛠 技術仕様開発言語: JavaScript (ES6+), HTML5 Canvas, CSS3レンダリング: requestAnimationFrame による 60FPS 描画物理・判定:ベクトル演算による距離・角度計算円形衝突判定（Circle Collision Detection）魔法弾のベクトルの反転による反射ロジック🚀 プレイ方法このリポジトリをダウンロードまたはクローンします。index.html を任意のブラウザで開きます。画面をクリックして、伝説の剣士としての戦いを始めてください。Bash# クローンして実行する場合
